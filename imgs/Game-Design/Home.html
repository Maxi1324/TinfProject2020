Game Design

Game Design ist der Prozess des Planens und entwerfen des Spiels. Beim Game Design wird entschieden ob das Spiel Spaß macht oder nicht.

Beim Game Designen werden die Mechaniken entworfen wie zb. bei Mario, dass man durch die Welt springt um zum Zeil zu kommen

Game Design hat ein absolut rießiges spektrum an Aufgaben zb. muss der Game Designer Beachten das das Spiel schwer ist, aber nicht zu schwer. Der Game Designer muss aufpassen das die Spiel Mechankiken an die Zielgruppe entsprechend angepasst sind ...
