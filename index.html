Worum geht es auf dieser Webseite

Auf dieser Webseite wird man über verschiedene Themen der Spieleentwicklung belehrt, wie zb. die Tools einige Techniken bzw. grundlegende dinge über Game dev, außerdem wird noch etwas über Game Design gesprochen.

Wenn es mu Spieleentwicklung geht ändert sich alles ziemlich schnell, egal ob es um die Tools oder standards geht

zb. bei den Tools, früher hat man meistens von ganz vorne begonnen, oder simple Engines, welche das Rendering oder so grundsätzliche sachen übernehmen, die Engines waren nur ein Stück code Heute sind Game Engines mehr Unity zumBeispiel hat ein Gui und einen haufen an integrierten Features.

Plattformen für Spiele

Es gibt verschiedene Plattformen um Spiele zu veröffentlichen. PCs Konsolen oder mit WebGl inzwischen im WebBrowser.
Früher ging es nicht für mehrere Plattformen gleichzeitig zu Entwickeln, heute geht es. Auf verschiedene Wege, über eine Game-Engine, welche das kann, oder durch den Browser, durch WebGl früher auch durch Flash heute allerdings nicht mehr der Support wurde beginn 2021 aufgegeben.

Programmieren überhaupt

Programmieren hat ein sehr weites Spektrum, um ein Spiel zu erschaffen braucht man allerdings mehr als nur Programmieren, man braucht Game-Design Grafiken bzw. 3D oder 2D artist.

Beim Programmieren gibt es verschiedene sachen zu beachten, man muss sich erstmal informieren welche Tools man nutzt bzw. welche Programmier sprache. Man soll auch aufpassen, dass man das Rad nicht neu erfindet.
